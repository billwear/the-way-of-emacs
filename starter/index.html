Don’t have Emacs? Try our Starter Kit.

Runs with an isolated config. No changes to your Emacs settings. One command to remove.

Download Starter Kit (.zip) SHA-256 How it works

Run it (pick your OS)

Windows (PowerShell)

  1. Download & unzip the Starter Kit.
  2. Open PowerShell in that folder.
  3. Run:
Unblock-File .\\start-emacs.ps1
.\start-emacs.ps1
First run fetches the official portable Emacs ZIP to %USERPROFILE%\\.twoe\\emacs-portable.

macOS (Terminal)

  1. Download & unzip the Starter Kit.
  2. Open Terminal in that folder.
  3. Run:
chmod +x start-emacs.sh
./start-emacs.sh
Best-effort DMG launch; if it fails, it will suggest brew install --cask emacs.

Linux (Terminal)

  1. Download & unzip the Starter Kit.
  2. Open a shell in that folder.
  3. Run:
chmod +x start-emacs.sh
./start-emacs.sh
If Emacs isn’t present, it prints the correct package-manager command for your distro.

What you’ll see

The kit launches Emacs with a separate init directory and opens a short 30‑minute tour. Your existing configuration isn’t touched.

Uninstall

Delete everything the kit created:

# macOS / Linux
./uninstall.sh

# Windows (PowerShell)
.\uninstall.ps1

This removes ~/.twoe (or %USERPROFILE%\\.twoe on Windows), where the kit stores its files.
