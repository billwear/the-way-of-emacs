30‑Minute Tour

Five short drills to make Emacs feel natural: windows & buffers, files & projects, search & replace, mark & region, and saving & quitting. Each drill is 4–6 minutes with real key chords and tiny Lisp to “make it yours”.

Start Drill 1 Printable cheatsheet 
M-x package-install RET which-key RET
;; Discover keybindings as you type:
(which-key-mode 1)

;; Switch to help quickly:
(global-set-key (kbd "C-h .") #'display-local-help)

Drill 1 — Windows & Buffers (≈6 min)

Navigate like a pro: split panes, switch focus, close, and list buffers.

  1. C-x 2 split window horizontally; C-x 3 vertically
  2. C-x o other-window (cycle focus)
  3. C-x 0 delete-window; C-x 1 delete-other-windows
  4. C-x b switch-to-buffer; C-x C-b list-buffers

Make it yours: quick cycle with C-`.

(global-set-key (kbd "C-`") #'other-window)
; Try this sequence:
C-x 2   C-x 3   C-x o   C-x o
C-x b *scratch* RET
C-x 0   C-x 1

Drill 2 — Files & Projects (≈6 min)

Open, save, recent files; jump across a project.

  1. C-x C-f find-file; C-x C-s save-buffer
  2. C-x C-r recentf-open-files (enable recentf)
  3. C-x p f project-find-file; C-x p s project-switch-project

Make it yours: enable recentf, set a handy key.

(recentf-mode 1)
(global-set-key (kbd "C-c r") #'recentf-open-files)
; Try this sequence:
C-x C-f tour.txt RET
Type a line, then: C-x C-s
C-x p s  ; choose a project
C-x p f  ; open a file in that project

Drill 3 — Search & Replace (≈7 min)

Incremental search, multi-hit navigation, and query-replace.

  1. C-s isearch-forward; C-r isearch-backward
  2. M-% query-replace; C-M-% query-replace-regexp
  3. During replace: y yes, n no, ! all

Make it yours: easy “repeat last search” with F3.

(global-set-key (kbd "") #'isearch-repeat-forward)
; Try this sequence:
C-s word RET   C-s C-s   M-% word RET term RET
Press y n ! as prompted

Drill 4 — Mark & Region (≈6 min)

Select, cut, copy, paste; expand/shrink region.

  1. C-SPC set-mark-command; move cursor to select
  2. M-w copy; C-w cut; C-y yank
  3. M-y cycle kill-ring after a yank

Make it yours: duplicate line/region command.

(defun twow/duplicate-line-or-region ()
  "Duplicate current line, or active region."
  (interactive)
  (let ((beg (if (use-region-p) (region-beginning) (line-beginning-position)))
        (end (if (use-region-p) (region-end)       (line-end-position))))
    (let ((text (buffer-substring beg end)))
      (goto-char end)
      (newline)
      (insert text))))
(global-set-key (kbd "C-c d") #'twow/duplicate-line-or-region)
; Try this sequence:
C-SPC  move to select  M-w
C-y   M-y M-y  ; cycle kills
C-c d ; duplicate

Drill 5 — Save, Quit & Help (≈5 min)

The confident exit and finding help fast.

  1. C-x C-s save-buffer; C-x s save-some-buffers
  2. C-x C-c save-buffers-kill-terminal
  3. C-h k describe-key; C-h f describe-function

Make it yours: safe quit prompts.

(setq confirm-kill-emacs 'y-or-n-p)
; Try this sequence:
C-x s   ; save multiple buffers
C-h k C-x C-c  ; what does quit do?
C-x C-c ; quit (answer y)
