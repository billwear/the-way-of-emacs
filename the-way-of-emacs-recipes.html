Task-based snippets with a short “Try it” flow and a tiny “Make it yours” customization. All examples stick close to GNU defaults and built-ins; optional packages are free software.

Navigation & Editing

Dired power moves (bulk rename)

Try it:

C-x d      ; open dired
C-x C-q    ; wdired-change-to-wdired-mode
C-c C-c    ; apply renames

Make it yours: handy key for toggling details.

(with-eval-after-load 'dired
  (define-key dired-mode-map (kbd "h") #'dired-hide-details-mode))

Rectangle editing (columns)

Try it: select a rectangle, then insert numbers.

C-SPC  move to select column block
C-x r N   ; rectangle-number-lines
C-x r t , ; string-rectangle (insert ",")

Make it yours: quick key for string-rectangle.

(global-set-key (kbd "C-c r t") #'string-rectangle)

Registers & bookmarks

Try it:

C-x r SPC a   ; point-to-register 'a'
C-x r j a     ; jump-to-register 'a'
C-x r m       ; bookmark-set
C-x r b       ; bookmark-jump

Make it yours: fast bookmark menu.

(global-set-key (kbd "C-c b") #'bookmark-bmenu-list)

Repeat change across lines

Try it: use keyboard macros.

F3    ; kmacro-start-macro
...   ; do edits on line 1
C-n   ; next-line
F4    ; kmacro-end-and-call-macro
C-u 10 F4 ; repeat 10 times

Make it yours: name your macro.

(call-last-kbd-macro 'my-macro)

Project grep (built-in)

Try it:

C-x p g   ; project-find-regexp
M-x rgrep ; recursive grep with prompts

Make it yours: prefer project grep by default.

(global-set-key (kbd "C-c /") #'project-find-regexp)

Edit remote files with TRAMP

Try it: open via SSH:

C-x C-f /ssh:user@host:/etc/hosts RET

Make it yours: keep connections alive.

(setq tramp-ssh-controlmaster-options
      "-o ControlMaster=auto -o ControlPath='tramp.%%C' -o ControlPersist=yes")

Version Control

Magit in 5 minutes

Try it:

M-x package-install RET magit RET
C-x g     ; magit-status
s         ; stage
c c       ; commit
P u       ; push (to upstream)

Make it yours: global binding.

(global-set-key (kbd "C-c g") #'magit-status)

Built-in VC (quick commit)

Try it:

C-x v v   ; do next logical VC step
C-x v =   ; diff
C-x v u   ; revert

Make it yours: diff on save (optional).

(add-hook 'after-save-hook #'vc-next-action)

Org Mode

Org capture everywhere

Try it:

(global-set-key (kbd "C-c c") #'org-capture)
(setq org-capture-templates
      '(("t" "Todo" entry (file "~/org/inbox.org")
         "* TODO %?\n  %U\n  %a")))

Make it yours: refiling destinations.

(setq org-refile-targets '(("~/org/next.org" :maxlevel . 2)
                            ("~/org/projects.org" :maxlevel . 2)))

Shell & Build

Compile & next-error

Try it:

M-x compile RET make test RET
M-g n   ; next-error
M-g p   ; previous-error

Make it yours: fast bindings.

(global-set-key (kbd "M-]") #'next-error)
(global-set-key (kbd "M-[") #'previous-error)

Eshell anywhere

Try it:

M-x eshell
C-x 3   ; split and keep shell view

Make it yours: toggle popup shell.

(defun twow/toggle-eshell ()
  (interactive)
  (if (get-buffer "*eshell*")
      (kill-buffer "*eshell*")
    (eshell)))
(global-set-key (kbd "C-c e") #'twow/toggle-eshell)
