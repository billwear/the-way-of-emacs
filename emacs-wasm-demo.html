Emacs (WASM) — Minimal Demo

Point this loader at an Emacs‑WASM build’s index.html (remote URL or local ./emacs/index.html).

How to use

  1. Paste a URL to an Emacs‑WASM index.html (or host one locally), then click Launch.
  2. If local, place the build next to this file as ./emacs/ and use ./emacs/index.html.

Keyboard tips: On macOS use ⌥ as Meta. If C‑SPC is blocked, use C‑@. Browsers may capture C‑w/C‑t.


If the frame stays blank, the URL might block embedding (X‑Frame‑Options). Open it in a new tab instead.

Where do I get a build?

Use any public Emacs‑to‑WebAssembly build (community “emacs‑wasm” releases). You’ll get a folder with index.html, a .wasm, and assets. Host it somewhere static (or put it in ./emacs/) and point above to its index.html.

This page is only a loader; it doesn’t contain binaries.
